# 视频提示词工业化生成工作流 V1.9.3

**版本**: 1.9.3  
**创建时间**: 2026-05-25  
**基于版本**: V1.9.2  
**定位**: Seedance 2.0 专属视频提示词生成工作流（文戏增强版：台词标点、情绪递进、微表情、声音表演）

---

## 版本升级说明

V1.9.3 在 V1.9.2 的工业化结构上新增“文戏表演引擎”。  
核心目标不是增加篇幅，而是提高 Seedance 2.0 在短剧文戏中的表演可控性：

1. 台词标点转化为停顿、重音、呼吸、尾音和眼神节拍。
2. 情绪从“标签”升级为“过程”：触发事件 → 认知反应 → 身体泄漏 → 压抑/伪装 → 失控点 → 情绪残留。
3. 每句台词绑定表情状态、声音状态和标点节奏。
4. 近景/特写镜头强化眼睛、嘴唇、下颌、喉咙、呼吸等微表情。
5. 负面词增加文戏表演失败约束，避免突然大哭、夸张瞪眼、全程喊叫、嘴型与台词节奏不匹配。
6. 新增“文戏降噪与影视化控制”：每个镜头只锁定少量关键锚点，避免 AIGC 被过密提示词稀释。

---

## 红线警告

### 红线 1：台词遗漏

**铁律**：收到剧本后，第一步必须逐句提取所有台词，编号统计，确认无遗漏。

**流程**：标记对话/心理/旁白 → 提取表格编号 1-N → 二次核对 → 统计字数时长 → 超长拆分 → 嵌入镜头。

**台词嵌入规则**：
- 台词直接嵌入对应镜头提示词。
- 用英文双引号 `""` 包裹。
- 禁止修改、遗漏、删减剧情。
- 原始标点必须保留，除非用户明确要求改写。

### 红线 2：文生图提示词缺失

生成视频提示词之前，必须先完成角色、场景、道具文生图提示词。

### 红线 3：每镜头必须标注画幅

每个镜头必须在时间码后标注画幅。

示例：

```text
【镜头 1】【00:00 - 00:05】（16:9）
```

### 红线 4：禁止旁白标注

禁止出现 `（旁白）`、`（心想）`、`（回忆）` 等标注。  
必须转化为镜头画面、声音或表演行为。

### 红线 5：空间连续性

同场景段落必须根据剧情推进更新人物位置，段首继承上段段尾位置。  
每组虽独立上传，但内部必须自洽，跨组也要有空间承接说明。

### 红线 6：空间站位禁止脑补

空间位置必须逐字对照原文，禁止添加原文未提及的方位词。

禁止事项：
- 禁止添加“左/右/前/后方”等原文未提及的方位词。
- 禁止添加原文未描述的视线关系。
- 禁止“合理脑补”。即使逻辑通顺，原文没写就不能加。

正确做法：

```text
原文："收银台后，林屿抬头"
正确：林屿（收银台后，抬头）
错误：林屿（收银台右后方，看向两人）
```

### 红线 7：人物移动路径必须写清

每个角色的位置变化必须包含“从 A 到 B”的过程描述。

正确做法：

```text
错误：小满手边放着热水。
正确：小满从收银台走到小桌旁，放热水到唐柚手边。
```

### 红线 8：防越轴检查表禁止只打勾

防越轴检查表每格必须填写具体描述，禁止只写“通过”“OK”“√”。

### 红线 9：分镜字段禁止两套标准混用

每个镜头必须且只能按“9+1 字段公式”填写，禁止简写、漏写字段或混用旧格式。

### 红线 10：负面词禁止照抄通用列表

每个剧本必须根据类型动态生成负面词，必须包含三层：

```text
Layer 1 基础负面词：所有类型通用。
Layer 2 类型专属负面词：根据剧本类型选配。
Layer 3 场景专属负面词：根据当前场景动态生成。
```

### 红线 11：台词标点不可忽略

台词中的 `，。！？？！……——～、` 必须被视为表演指令。  
标点决定停顿、重音、呼吸、尾音、眼神变化和情绪泄漏点。

禁止事项：
- 禁止删除、替换或合并原台词标点。
- 禁止把省略号当普通停顿处理。
- 禁止把问号写成平铺直叙。
- 禁止把感叹号全部处理成大喊。

### 红线 12：文戏情绪禁止标签化

禁止只写“悲伤、愤怒、崩溃、心动、委屈”。  
必须写成可拍、可听、可生成的过程：

```text
触发事件 → 第一反应 → 身体泄漏 → 压抑/伪装 → 失控点 → 情绪残留
```

### 红线 13：脸、声、词必须同步

每句台词必须至少绑定三项：

```text
表情状态 + 声音状态 + 标点节奏
```

禁止“脸在崩溃大哭，声音却稳定冷静”这类冲突，除非剧情明确是伪装，并且写清伪装机制。

### 红线 14：文戏提示词禁止过载

文戏镜头不是把所有表情词堆满。每个文戏镜头必须执行“1+3+1+1”重点控制：

```text
1 个核心情绪机制
最多 3 个微表情锚点
1 个声音节奏重点
1 个镜头语言目的
```

禁止事项：
- 禁止同一镜头同时塞入 5 个以上微表情变化。
- 禁止每个镜头都写完整 15 秒情绪弧线。
- 禁止负面词超过当前组真实风险，导致重点稀释。
- 禁止把声音表演当成视频模型强约束；声音细节主要服务嘴型节奏、配音和后期。

## Step 1：需求确认（P0）

| 确认项 | 选项 |
|:---|:---|
| 画幅比例 | 9:16 / 16:9 / 21:9 / 2.35:1 |
| 视觉风格 | 写实电影感 + 现代都市 / 古装 / 悬疑 / 家庭 / 其他 |
| 文戏强度 | 低克制 / 中等情绪 / 高压对峙 / 崩溃边缘 |
| 台词处理 | 完整保留 / 用户允许润色 / 无台词纯表演 |

默认设置：
- 每组最长 15 秒。
- 每组独立上传，时间码从 `00:00` 开始。
- 文戏镜头优先使用稳定机位、近景、特写、中近景。
- 台词多于 45-55 个汉字时，优先拆组，不强行压缩语速。

## Step 2：台词逐句提取（P0）

收到剧本后，先提取台词，再写提示词。

台词提取表：

```markdown
| 编号 | 角色 | 原台词（保留标点） | 字数 | 语速类型 | 预计时长 | 所属组 | 所属镜头 |
|:---:|:---|:---|---:|:---|---:|:---:|:---:|
```

语速参考：

| 类型 | 语速 | 适用场景 |
|:---|:---:|:---|
| 舒缓抒情温柔对话 | 3.0 字/秒 | 告白、安慰、柔和回忆 |
| 沉思安静内心独白 | 3.2 字/秒 | 心理独白、迟疑 |
| 默认正常情节对白 | 4.4 字/秒 | 日常对话 |
| 剧情推进对白 | 4.6 字/秒 | 信息推进 |
| 客观叙事旁白 | 4.8 字/秒 | 无强情绪旁白 |
| 紧张对峙激烈冲突 | 5.0 字/秒 | 争执、逼问 |
| 慌乱惊恐急促情绪 | 5.5 字/秒 | 逃跑、惊恐 |

台词时长是镜头时长的硬下限：

```text
镜头时长 >= 台词字数 / 语速 + 必要停顿 + 表情反应时间
```

文戏额外要求：
- 不要把 15 秒全部填满台词，必须留出沉默、反应、眼神变化。
- 含省略号、破折号、问号的台词，必须额外预留停顿和尾音时间。

## Step 2.5：台词标点与语音表演拆解（V1.9.3 新增，P0）

每句台词必须做标点解析。  
标点不是排版符号，而是演员表演节拍。

标点规则：

| 标点 | 表演含义 | 提示词写法 |
|:---:|:---|:---|
| `，` | 小停顿、试探、思考、忍住情绪 | 停顿 0.2-0.5 秒，眼神轻微变化 |
| `。` | 情绪落地、克制、冷淡、最终确认 | 尾音下落或收住，表情停留 |
| `！` | 重音加压、警告、震惊、爆点 | 指定重音词，避免全程喊叫 |
| `？` | 怀疑、试探、脆弱追问 | 尾音轻微上扬，眼神寻找反应 |
| `？！` | 质问加爆点 | 只用于强烈指控或震惊，声压短促上升 |
| `……` | 情绪卡在喉咙里 | 呼吸中断、吞咽、嘴唇微张后停住 |
| `——` | 转折、打断、真相延迟、心理断裂 | 前半句被截断，后半句换气或变声线 |
| `～` | 柔软、亲近、撒娇、轻松尾音 | 尾音拉软，严肃文戏慎用 |
| `、` | 并列节奏、压迫式列举 | 每个并列词轻压重音 |

台词标点拆解表：

```markdown
| 编号 | 原台词 | 标点节奏 | 潜台词 | 声音行为 | 表情锚点 |
|:---:|:---|:---|:---|:---|:---|
```

声音行为公式：

```text
音量 + 语速 + 呼吸 + 停顿位置 + 重音位置 + 尾音
```

示例：

```text
"你真的……从来没信过我吗？"
省略号处喉咙卡住 0.5 秒，"从来"轻重音，问号尾音上扬后迅速收住；
声音变轻、气息不稳，眼神先停住再向下偏移，嘴唇微张但没有立刻继续说。
```

## Step 3：空间状态追踪（P0）

流程：
1. 通读原文，识别所有场景。
2. 逐句标记原文中的位置描述。
3. 记录每个角色初始位置，并附原文依据。
4. 标记剧情推进中的走位，并附原文依据。
5. 按段落拆分，记录段首/段尾位置。
6. 建立段落间位置传递链。
7. 填写防越轴检查表，每格写具体描述。

空间状态追踪表：

```markdown
| 段落 | 场景 | 角色 A 位置（原文依据） | 角色 B 位置（原文依据） | 位置变化 |
|:---:|:---|:---|:---|:---|
```

防越轴检查表：

```markdown
| 镜头 | 角色位置关系 | 视线/朝向依据 | 摄影机轴线 | 移动过程 | 是否越轴风险 |
|:---:|:---|:---|:---|:---|:---|
```

## Step 4：抽象到具象转译（P0）

核心拷问：

```text
这个描写，摄影机拍得到吗？
麦克风录得到吗？
演员能用身体哪一部分演出来？
```

不可拍的心理词必须转成可见、可听的行为。

转译示例：

| 抽象描写 | 具象转译 |
|:---|:---|
| 内心崩溃 | 眼神失焦，嘴唇微张，呼吸停一拍，手指失去力气 |
| 强忍泪水 | 下眼睑发红，鼻尖微红，嘴唇压紧不动 |
| 心动 | 视线短暂下移再上移，呼吸慢半拍，指尖无意识摩挲杯沿 |
| 故作冷静 | 声音过稳，手指节奏比平时快，眼神不敢久停 |
| 愤怒压住 | 下颌收紧，语速变慢，最后几个字压得更重 |
| 迟来的理解 | 先无反应，眼神停住，嘴唇轻微分开，呼吸变浅 |

## Step 4.5：文戏表演弧线（V1.9.3 新增，P0）

文戏不能直接写结果情绪，必须写情绪进入脸、呼吸、声音和身体的过程。

通用机制：

```text
触发事件 → 第一认知反应 → 身体泄漏 → 压抑/伪装 → 失控点 → 情绪残留
```

15 秒文戏默认时间轴：

```text
00:00-00:01 控制住的脸，呼吸稳定。
00:01-00:03 触发信息进入，眼神先停。
00:03-00:05 理解发生，嘴唇、下眼睑或呼吸开始变化。
00:05-00:07 情绪进入身体，喉咙吞咽或下颌收紧。
00:07-00:09 角色试图压住，声音变轻、变慢或变硬。
00:09-00:11 第一次失控，停顿、颤音或眼眶湿润。
00:11-00:13 情绪不可否认，但动作仍克制。
00:13-00:15 情绪残留，不要只停在一个表情姿势。
```

短时长文戏：

| 时长 | 表演弧线 |
|:---:|:---|
| 5 秒 | 中性 → 触发 → 第一泄漏 → 残留 |
| 8 秒 | 中性 → 触发 → 认知 → 压住 → 可见裂缝 |
| 10 秒 | 中性 → 触发 → 反应 → 压住 → 第一次失控 → 残留 |

微表情字段库：

| 部位 | 可用描述 |
|:---|:---|
| 眼睛 | 聚焦、失焦、变亮、变冷、湿润、躲闪、停住、下移 |
| 下眼睑 | 发红、轻颤、绷紧、微微抬起 |
| 眉 | 眉峰压低、眉尾松开、短暂皱眉后恢复 |
| 嘴唇 | 抿紧、微张、颤动、笑意失败、尾音压住 |
| 下颌 | 收紧、放松失败、咬肌轻动 |
| 喉咙 | 吞咽、卡住、说话前换气 |
| 呼吸 | 变浅、停一拍、变重、破碎、被笑或哭打断 |
| 身体 | 肩膀微僵、指尖收紧、动作慢半拍 |

## Step 4.6：文戏降噪与影视化控制（V1.9.3 强化，P0）

文戏提示词必须从“全量控制”转为“重点控制”。  
AIGC 更容易执行清晰的少量锚点，而不是密集的表情清单。

### 1+3+1+1 镜头控制公式

每个文戏镜头生成前，先确定：

```markdown
| 项目 | 限制 | 示例 |
|:---|:---|:---|
| 核心情绪机制 | 只选 1 个 | 迟来的理解 / 压住的愤怒 / 假笑崩塌 |
| 微表情锚点 | 最多 3 个 | 眼神停住 + 喉咙吞咽 + 嘴唇压紧 |
| 声音节奏重点 | 只选 1 个 | 省略号卡住 / 问号尾音上扬 / 句号压低落地 |
| 镜头语言目的 | 只选 1 个 | 不切走让观众看见压抑 / 用前景遮挡制造隔阂 |
```

### AIGC 强弱约束分层

写提示词时必须区分哪些内容是视频模型强约束，哪些是配音/后期参考。

```markdown
| 层级 | 内容 | 写法 |
|:---|:---|:---|
| 强约束 | 人物、场景、镜头、景别、动作、脸部可见变化、嘴型节奏 | 必须写进镜头主体动作和关键约束 |
| 中约束 | 呼吸、停顿、眼神、手指小动作、表演节奏 | 写成 2-3 个可见锚点 |
| 弱约束/后期参考 | 音量、重音、尾音、声线质感 | 写入音频规范，不强行当画面约束 |
```

### 文戏机制选择表

避免所有文戏都套同一条“触发→失控”公式。根据戏剧功能选择机制：

| 机制 | 情绪走向 | 适用场景 | 风险 |
|:---|:---|:---|:---|
| 压抑型 | 越痛越安静 | 告别、亲情、旧伤 | 避免空白无变化 |
| 反讽型 | 嘴在笑，眼已冷 | 背叛、失望、体面分手 | 避免假笑过度 |
| 延迟型 | 当场无反应，最后一秒才泄漏 | 真相揭示、噩耗 | 避免前段太木 |
| 权力型 | 谁先动情，谁先输 | 对峙、谈判、复仇 | 避免反派邪笑 |
| 空白型 | 表情很少，呼吸和停顿暴露一切 | 创伤、麻木、极端疲惫 | 避免毫无递进 |
| 回暖型 | 防御慢慢松开 | 和解、安慰、久别重逢 | 避免突然亲密 |

### 节奏留白规则

文戏关键台词后必须留出反应时间。  
以下情况必须预留 `0.5-2 秒` 留白：

- 问句之后：看对方反应，尾音不要立刻切走。
- 省略号之后：允许喉咙、呼吸、眼神完成停顿。
- 真相揭示之后：先让角色不反应，再让理解进入眼睛。
- 告别句之后：让句号落地，留情绪残留。
- 道歉/感谢之后：不要马上用下句覆盖。

### 镜头语言目的表

每个文戏镜头必须有一个影视目的，不能只说“近景拍脸”。

| 目的 | 镜头策略 |
|:---|:---|
| 让观众看见压抑 | 固定特写，不切走，保留呼吸 |
| 表达关系隔阂 | 前景遮挡、门框/玻璃/桌面隔开人物 |
| 表达权力压迫 | 低幅度推近、视线高度略低或略高 |
| 表达孤立 | 单人置于画面偏侧，留出空白空间 |
| 表达迟来的理解 | 先中近景稳定，再极慢推向眼睛 |
| 表达体面崩塌 | 正常构图逐渐变成更紧的脸部取景 |
| 表达未说出口 | 切在嘴唇微张前后，但不让角色说完 |

## Step 5：文生图提示词生成（P0）

生成视频提示词前，必须完成角色、场景、道具文生图提示词。

### 角色文生图提示词

必须包含：

1. 年龄、身份、职业、地域气质。
2. 脸型、骨相、五官结构。
3. 1-2 个核心视觉记忆点。
4. 眼睛、眉毛、鼻子、嘴唇、发型。
5. 皮肤质感、生活痕迹。
6. 服装、饰品、职业道具。
7. 情绪气质，但不写夸张表演。

前置格式：

```text
比例 21:9，真人写实风格，右侧角色头部高清特写，左侧角色全身三视图（正面/侧面/背面），中性全局照明，纯白无缝背景，8K 超高清分辨率。
```

### 场景文生图提示词

必须包含：

1. 时间、天气、地点。
2. 空间结构和可见动线。
3. 光源位置和色温。
4. 关键道具。
5. 场景质感、年代感、生活痕迹。
6. 可用于镜头承接的固定视觉符号。

### 道具文生图提示词

必须包含：

1. 尺寸和比例参照。
2. 材质、磨损、使用痕迹。
3. 与人物身份/职业/剧情的关系。
4. 是否需要在近景中保持一致。

## 核心规范

### 0：组结构定义

组是最小上传单元。每组独立生成、独立上传到 Seedance。  
AI 不感知前后组，因此每组必须自包含。

| 属性 | 规则 |
|:---|:---|
| 时长 | 每组不超过 15 秒 |
| 时间码 | 每组从 `00:00` 开始 |
| 自包含 | 重复必要的人物、场景、道具、情绪残留、台词、声音、负面词 |
| 结构 | 单镜到底或 2-4 个镜头快切 |

每组输出结构：

```markdown
### 第 N 组：组标题
**空间状态**：...
**情绪状态**：继承/当前触发/目标残留

【全局风格与环境】
...

【镜头 1】【00:00 - 00:XX】（画幅）9+1 字段

【关键约束】
空间线 - ...
光线线 - ...
运镜线 - ...
画面线 - ...
风格线 - ...
表演线 - ...
台词线 - ...

【音频规范】
全局音轨：...
逐镜头声音设计：...
逐句台词声音表演：...

【负面词】
Layer 1 基础负面词：...
Layer 2 类型专属负面词：...
Layer 3 本组专属负面词：...

视频时长 XX 秒
```

### 1：全局风格与环境

输出格式：

```markdown
【全局风格与环境】
时间：{早晨/午后/黄昏/深夜}
天气：{晴/雨/雪/室内}
氛围：{压抑/温暖/紧张/梦幻/冷峻}

电影质感：{cinematic / realistic / 3DCG} | 胶片：{35mm 中颗粒 / ARRI / UE5} | 画幅：{16:9 / 9:16 / 2.35:1}
布光方式：{主光位置 + 光质 + 光比}
色调：{主调} | 色彩处理：{通道偏移/色相隔离/色温逻辑}
色彩主题：{主色调 + 辅色调 + 情绪色}
关键视觉符号：{0-2 个反复出现的视觉元素}
人物面部质感：{肤色 + 高光 + 纹理 + 质地}
拒绝项：{不要 X, NOT Y}
```

### 2：镜头 9+1 字段公式

每个镜头必须且只能按以下顺序填写：

```text
【镜头 N】【时间码】（画幅）
1 视角/景别 +
2 机位 +
3 焦段/光圈 +
4 运镜 +
5 主体动作（含台词 + 标点节奏 + 声音表演 + 微表情） +
6 环境细节 +
7 构图/取景 +
8 焦点/景深 +
9 画质标识 +
10 光线（仅当本镜头偏离全局光线时填写）
```

V1.9.3 文戏降噪要求：
- 字段 5 只能锁定 1 个核心情绪机制。
- 字段 5 最多写 3 个微表情锚点。
- 声音细节只保留 1 个最重要的标点/节奏重点。
- 镜头语言目的写入字段 7 或关键约束“画面线/表演线”。

字段 5 文戏增强写法：

```text
角色动作 + 原台词 + 标点节奏 + 声音行为 + 微表情变化
```

示例：

```text
唐柚抬眼看向林屿，先吸一口气，嘴唇抿紧后松开，用很轻但压住颤音的声音说："你真的……从来没信过我吗？"；省略号处喉咙卡住 0.5 秒，"从来"轻重音，问号尾音上扬后迅速收住，眼眶湿润但不落泪
```

### 3：精确时间码

每组时间码从 `00:00` 开始，禁止跨组累计。

多镜头组：

```text
【镜头 1】【00:00 - 00:06】（16:9）
【镜头 2】【00:06 - 00:12】（16:9）
【镜头 3】【00:12 - 00:15】（16:9）
```

单镜到底：

```text
【镜头 1】【00:00 - 00:15】（16:9）单镜 slow push
  - 00:00-00:03 阶段描述
  - 00:03-00:07 阶段描述
  - 00:07-00:11 阶段描述
  - 00:11-00:15 阶段描述
```

### 4：文戏镜头策略

文戏优先策略：

| 场景 | 推荐镜头 |
|:---|:---|
| 克制告别 | 固定近景 / slow push 特写 |
| 压抑对峙 | 过肩近景 / 中近景反打 |
| 迟疑告白 | 眼平近景 / 微推 |
| 崩溃边缘 | 稳定特写 / 极慢推近 |
| 冷静威胁 | 固定中近景 / 低幅度推近 |
| 沉默反应 | 特写 / 不切走 / 保留呼吸 |

文戏镜头的影视目的必须先于镜头形式。  
禁止为了“好看”随机切特写，也禁止没有目的地多机位覆盖。

镜头目的选择：

| 戏剧目的 | 推荐处理 | 禁止 |
|:---|:---|:---|
| 看见压抑 | 固定近景或特写，保留 1-2 秒沉默 | 快切打断表演 |
| 制造隔阂 | 前景遮挡、门框、玻璃、桌面分隔 | 无依据改变站位 |
| 强化权力 | 稳定中近景，轻微高低视角 | 夸张仰拍/俯拍 |
| 孤立人物 | 人物偏画面一侧，留空白 | 所有镜头都居中对称 |
| 真相落地 | 先不反应，再慢推到眼睛 | 立刻哭喊 |
| 关系回暖 | 景别略放松，呼吸变顺 | 突然拥抱或亲密动作 |

文戏限制：
- 不要随机切走环境空镜。
- 不要用大幅度运镜稀释表演。
- 不要让演员频繁转身、走动、拿道具。
- 微表情重要时，镜头必须稳定，焦点必须锁脸。
- 重要台词后必须留反应，不要立刻切到下一句。
- 每个镜头只服务一个戏剧目的。

### 5：运镜衔接 5 模式

相邻镜头运镜必须能归类：

| 模式 | 定义 | 适用 |
|:---|:---|:---|
| 同向衔接 | 两镜运动方向一致，速度匹配 | 追逐、连续动作 |
| 反向衔接 | 第二镜用相反方向回应第一镜 | 对话、对峙 |
| 静切动 | 固定切入运动 | 情绪转折 |
| 动切静 | 运动切入固定 | 震惊、发现 |
| 同质叠加 | 相同运镜方式递进 | 仪式感、情绪积累 |

文戏优先使用：
- 固定 → 固定
- 固定 → 极慢推
- 极慢推 → 固定
- 反打镜头保持景别和焦段逻辑一致

### 6：关键约束区块

每组末尾必须输出关键约束区块。V1.9.3 增加“表演线”和“台词线”。

格式：

```markdown
【关键约束】
空间线 - {空间继承约束，逐条描述}
光线线 - {光线约束，逐条描述}
运镜线 - {运镜约束，逐条描述}
画面线 - {取景/构图/可见性约束，逐条描述}
风格线 - {风格/通用约束，逐条描述}
表演线 - {表情递进、微表情、身体泄漏、情绪残留}
台词线 - {原台词、标点、停顿、重音、尾音约束}
```

规则：
- 每一层都必须出现。
- 无约束时写“无特殊约束”，不能省略层级。
- 每条约束必须以“必须 / 禁止 / NOT / 保持 / 避免”开头。

## 音频规范

每组必须包含：

1. 全局音轨。
2. 逐镜头声音设计。
3. 逐句台词声音表演。

声音表演模板：

```text
角色因为{触发事件}情绪变化，声音{变轻/变低/变亮/更稳/更空}，
语速{变慢/变快/断续}，呼吸{变浅/变重/不稳/卡住}，
在"{关键词}"后停顿，重音落在"{关键词}"，
尾音{上扬/下落/压住/发空/颤动}，说："原台词"
```

逐句台词声音表演表：

```markdown
| 台词编号 | 原台词 | 音量 | 语速 | 呼吸 | 停顿 | 重音 | 尾音 | 面部同步 |
|:---:|:---|:---|:---|:---|:---|:---|:---|:---|
```

## 负面词规范

负面词必须三层完整。
每组负面词要短而准，优先保留当前最容易翻车的 5-8 条；不要把所有负面词机械堆满。

基础负面词：

```text
避免手部畸形，避免脸部变形，避免比例失调，避免透视错误，避免模糊失焦，避免噪点过多，避免过度磨皮，避免字幕，避免水印，避免多余角色，避免随机镜头运动。
```

文戏类型专属负面词：

```text
避免突然大哭，避免表情一步到位，避免夸张瞪眼，避免全程喊叫，避免嘴型与台词节奏不匹配，避免假笑过度，避免无情绪递进的空白表情，避免偶像剧式夸张肢体动作，避免情绪跳变。
```

标点专属负面词：

```text
避免忽略省略号停顿，避免问句尾音无变化，避免感叹号变成失控尖叫，避免破折号没有中断感，避免逗号没有真实换气，避免句号没有落点，避免顿号并列节奏混乱。
```

场景专属负面词示例：

| 类型 | 负面词 |
|:---|:---|
| 家庭文戏 | 避免煽情过度，避免立刻拥抱，避免哭喊式和解 |
| 爱情文戏 | 避免突然亲吻，避免油腻挑逗，避免直白偶像剧表情 |
| 对峙背叛 | 避免开场大吼，避免反派邪笑，避免大幅度摔东西 |
| 沉默反应 | 避免空白无变化，避免切走关键表情，避免添加无关台词 |
| 崩溃边缘 | 避免开场直接崩溃，避免泪水过早落下，避免五官夸张扭曲 |

情绪类型专属失败模式：

| 情绪类型 | 最容易失败 | 必须约束 |
|:---|:---|:---|
| 哭戏 | 太早落泪、五官扭曲、哭得太满 | 避免开场掉泪，泪水只作为后段失控或残留 |
| 怒戏 | 全程吼叫、动作过大 | 声音压低变慢，重音加压，不靠大喊表达 |
| 心动 | 油腻、偶像剧化、过度甜 | 只用眼神停顿、呼吸慢半拍、指尖小动作 |
| 威胁 | 邪笑、脸谱化反派 | 表情更静，尾音下压，避免露齿笑 |
| 告别 | 煽情过度、台词太满 | 句号落地后留白，动作少，眼神残留 |
| 和解 | 突然拥抱、情绪转暖过快 | 防御先松一点，不立刻亲密 |
| 麻木 | 演成面瘫 | 表情少但呼吸、眼神焦点、嘴唇有微小变化 |

负面词降噪公式：

```text
本组只保留：基础质量 2-3 条 + 类型失败 2-3 条 + 标点/表演风险 1-2 条。
```

## 自检清单

AI 内部核对，缺一不可。

| # | 检查项 | 通过条件 |
|:---:|:---|:---|
| 1 | 台词完整 | 每句台词已编号并嵌入对应镜头 |
| 2 | 标点保留 | 原台词标点未删除、未替换、未合并 |
| 3 | 标点表演 | 逗号/句号/问号/省略号等已转化为停顿、重音、尾音 |
| 4 | 台词时长 | 镜头时长 >= 台词时长 + 停顿 + 反应时间 |
| 5 | 画幅标注 | 每个镜头时间码后有画幅 |
| 6 | 9+1 字段 | 每镜头至少 9 字段，偏离全局光线时 10 字段 |
| 7 | 文戏弧线 | 情绪有触发、反应、泄漏、压抑、失控点、残留 |
| 8 | 微表情 | 眼睛、嘴唇、下颌、喉咙、呼吸至少出现 3 类 |
| 9 | 声画同步 | 声音状态与面部状态不冲突 |
| 10 | 空间连续 | 位置、朝向、走位有原文依据 |
| 11 | 无空间脑补 | 未添加原文没有的方位词 |
| 12 | 移动路径 | 位置变化写清“从 A 到 B” |
| 13 | 运镜合理 | 相邻镜头可归入 5 种衔接之一 |
| 14 | 构图边界 | 构图字段包含取景边界 |
| 15 | 负面词三层 | 基础 + 类型 + 本组专属完整 |
| 16 | 文戏负面词 | 已限制突然大哭、夸张表情、嘴型错位 |
| 17 | 音频规范 | 有全局音轨、逐镜头声音、逐句台词声音 |
| 18 | 组包合规 | 每组 <=15 秒，时间码组内归零 |
| 19 | 自包含 | 每组含人物、场景、道具、情绪状态、台词、约束 |
| 20 | 输出文件 | 最终写入 `.md` 文件，不在对话中完整铺开成片提示词 |
| 21 | 文戏降噪 | 每镜头符合 1+3+1+1，未堆满表情清单 |
| 22 | 影视目的 | 每个文戏镜头有明确镜头语言目的 |
| 23 | 留白 | 问句、省略号、真相揭示、告别句后有 0.5-2 秒反应 |
| 24 | 负面词降噪 | 本组负面词优先当前风险，未机械堆满 |

## V1.9.3 极简执行口诀

```text
先抽台词，保留标点；
先算时长，再拆组包；
先定空间，再写镜头；
情绪写过程，不写标签；
标点变停顿、重音、呼吸、尾音；
脸、声、词同步；
每镜头 1 个机制、3 个锚点、1 个声音重点、1 个镜头目的；
关键台词后留白；
每组自包含，15 秒内归零；
最后三层负面词和自检表收口。
```
